欢迎来到 **宝石民主 (Gemocracy)** 的官方 GitHub 组织 - 这是一个革命性的 Minecraft 生存多人服务器，玩家通过独特的宝石治理系统实现自治。

## 🌟 About Gemocracy / 关于宝石民主

Gemocracy redefines Minecraft server governance by eliminating traditional admin systems. Instead, power is distributed through physical gems that players can find, steal, and use to enforce their will on the server.

宝石民主通过消除传统的管理员系统，重新定义了 Minecraft 服务器治理。相反，权力通过玩家可以找到、偷取和使用的物理宝石进行分配。

### Core Philosophy / 核心理念

- **Physical Power**: Server permissions are forged into physical items\
  **物理权力**：服务器权限被铸造成物理物品
- **Eternal Revolution**: Power cycles through the player base\
  **永恒革命**：权力在玩家群体中循环
- **Player Sovereignty**: You are the rules - stability is impossible, revolution is inevitable\
  **玩家主权**：你就是规则 - 稳定是不可能的，革命是必然的

## 🎮 Server Features / 服务器特色

### The 7 Gems System / 七大宝石系统

Players can find and use powerful gems to gain temporary admin abilities:
玩家可以找到并使用强大的宝石来获得临时的管理员能力：

- **Justice Gem** - Authority to punish players\
  **正义宝石** - 惩罚玩家的权力
- **Truth Gem** - Access to CoreProtect for investigation\
  **真理宝石** - 访问 CoreProtect 进行调查
- **Flight Gem** - Temporary flight abilities\
  **飞行宝石** - 临时飞行能力
- **Life Gem** - Healing powers\
  **生命宝石** - 治疗能力
- **Assassin Gem** - Ability to appoint assassins\
  **刺客宝石** - 任命刺客的能力
- **Land Gem** - Land sovereignty management\
  **土地宝石** - 土地主权管理
- **Navigation Gem** - Compass tracking to locate other gems  
  **导航宝石** - 追踪其他宝石的位置

### Unique Gameplay / 独特玩法

- Empire building with land claims\
  通过土地声明建立帝国
- Player-built underground metro system\
  玩家建造的地下地铁系统
- Raw survival experience with no teleportation commands\
  原始生存体验，无传送命令
